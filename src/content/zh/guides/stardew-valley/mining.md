## 矿洞：农场以外最重要的系统

矿洞看起来只是「打怪挖矿」，但它其实决定了你整个农场的进展速度：铜矿升级工具→铁矿建造机器→金矿解锁更好设施→通关后的骷髅洞窟产出铱矿（用于铱锄、铱洒水器）。

越快通关矿洞，农场效率提升越快。

---

## 矿洞基础

### 进入矿洞

矿洞入口在农场东北方向（山间湖旁边），游戏第5天地震解锁。初次进入时，挖矿师杜沃会给你发信，并赠送炸弹。

### 电梯系统

每通过5层（5/10/15/20…），解锁对应的电梯层。之后从电梯可以直接跳到已解锁的最低楼层，避免重复下同样的层。

**重要**：电梯层必须在该层**找到并走下楼梯**才能解锁。只是到达这一层，不走楼梯离开，不会解锁电梯。

### 楼梯的生成规律

每一层都有随机生成的楼梯，有时在地图上清晰可见，有时需要清除石头或击败怪物才会出现。

**楼梯出现触发条件**（满足任意一个即可触发随机刷新）：
- 击败该层的所有怪物
- 清除该层所有石头
- 击败某些特定怪物后随机掉落

**技巧**：看到石头密集的房间，用炸弹清除——有高概率出现楼梯。

## 各区段矿石与策略

### 第1~39层：铜矿区

**主要产出**：
- 铜矿石（熔炼→铜锭）
- 石英（社区中心地质学家捆绑包）
- 大地水晶（社区中心用）
- 洞穴胡萝卜（觅食）

**主要怪物**：绿色史莱姆、臭虫

**战斗难度**：★☆☆☆☆（最容易）

**策略**：这个区段相对轻松。重点是收集足够的铜锭（3个铜锭可升级工具），同时找楼梯向下推进。不必清场，看到楼梯直接走。

---

### 第40~79层：铁矿区

**主要产出**：
- 铁矿石（熔炼→铁锭）
- 冰冻泪晶（社区中心地质学家捆绑包）
- 冰冻晶洞（社区中心用）

**主要怪物**：蓝色史莱姆、蝙蝠、幽灵

**战斗难度**：★★☆☆☆

**特别注意**：
- **蝙蝠**会在空中飞行，需要长按攻击键预判位置
- **幽灵**（第51层+）会穿过墙壁，避免被夹击
- 第45~65层冰雪主题区，地面有时会减速移动

**策略**：铁锭用量大（铁炉灶、发酵桶、大量机器），多收集铁矿石。不急着推进到第80层，先把背包填满再走。

---

### 第80~120层：金矿区

**主要产出**：
- 金矿石（熔炼→金锭）
- 火焰石英（社区中心地质学家捆绑包）
- 火石英和其他宝石
- 钻石（稀有，矿洞各层均有，80+层概率稍高）

**主要怪物**：红色史莱姆、岩石螃蟹、火球怪（方块怪）、影子野兽

**战斗难度**：★★★☆☆

**特别注意**：
- **岩石螃蟹**外观像石头，静止时难以区分，靠近才会动
- **火球怪**（方块形）只能从侧面或背面攻击，正面有护盾
- **影子野兽**（80层+）速度快，但击败它可获得虚空精华（社区中心用）

**策略**：这个区段是很多玩家第一年回来最多的区段，因为金锭用量极大（各种升级都需要金锭）。找到金矿洞后慢慢挖，不急着往下走。

## 通关矿洞的三种策略

### 策略1：稳扎稳打（推荐新手）

每天都进入矿洞，从已解锁的最低电梯层开始，争取每天向下推进5~10层。这种方式最安全：食物消耗少，装备需求低，不会因为提前耗尽体力或血量而损失物品。

**时间预期**：大约10~15天游戏内时间（约2~3周）完成全120层。

### 策略2：炸弹快攻（推荐中期玩家）

大量制作炸弹（铁锭×1+金矿石×1+蝙蝠翅膀×1，或向杜沃购买）。每一层用炸弹清除密集石头区域，快速找楼梯。

**适合条件**：铁矿石储量充足，可以大量制作炸弹。

**时间预期**：4~7天游戏内时间可通关。

### 策略3：楼梯全绕（最快但消耗最多）

制作大量木头楼梯（石头×99→楼梯）带入矿洞，遇到困难的层直接放楼梯跳过。

**适合条件**：有大量石头储备，追求最快速度通关（想尽早进入骷髅洞窟）。

**缺点**：大量跳过层数意味着错过很多矿石和宝石，物资积累更慢。

## 关键物品清单

### 必带物品

| 物品 | 数量 | 用途 |
|---|---|---|
| 煎蛋 / 浆果 | 10~15个 | 补充体力和血量 |
| 炸弹 | 5~10个 | 快速清石、对多个怪物造成伤害 |
| 木头楼梯 | 5~10个（可选） | 紧急跳过困难层 |
| 火炬 | 20个 | 点亮黑暗区域（矿洞里部分区域很暗） |

### 武器选择

**第1~39层**：初始剑（游戏赠送的普通剑）足够  
**第40~79层**：建议升级到铁制武器（铁锭×3可在市长办公室旁的宝箱获得一把剑，或购买）  
**第80~120层**：银匠铺购买金属剑，或者在矿洞宝箱里找到更好的武器

**武器比较**：
- 剑：标准全面，不错的选择
- 匕首：攻速快，单次伤害低，连击效果好
- 锤：单次伤害高，速度慢，建议有把握命中时使用

## 矿洞宝箱

矿洞每10层（10/20/30…）一定有一个宝箱，里面包含固定物品：

| 层数 | 宝箱内容 |
|---|---|
| 第10层 | 皮鞋 |
| 第20层 | 石灰岩地板 |
| 第40层 | 铁矿石 ×5 |
| 第70层 | 钢制小剑 |
| 第80层 | 靴子 |
| 第100层 | 鱼烟熏鱼 |
| 第110层 | 解锁骷髅洞窟地图提示 |
| 第120层 | **植物猪** + 解锁骷髅洞窟 |

第120层的宝箱包含一头特殊的「植物猪」，并触发「矿洞通关」事件，同时解锁沙漠骷髅洞窟的入口。

## 骷髅洞窟：进阶挖矿

完成矿洞后开放骷髅洞窟，位于沙漠（需先完成地窖捆绑包修好公共汽车）。

### 与矿洞的区别

| 特征 | 矿洞 | 骷髅洞窟 |
|---|---|---|
| 层数 | 120层固定 | 无限（越深越好） |
| 主要矿石 | 铜/铁/金 | **铱矿石** |
| 怪物强度 | 中等 | 高（需要好装备） |
| 节奏 | 适合日常探索 | 适合专门探索日 |

### 骷髅洞窟的价值

**铱矿石**是游戏内最稀有的金属：
- 铱锄：一次翻耕所有周围地块（革命性效率）
- **铱洒水器**：覆盖5×5区域（24块地），是游戏内最重要的效率升级之一
- 铱锭也用于制作其他高级工具和机器

### 骷髅洞窟攻略要点

1. **星期日或假日前往**：一整天不间断地探索，目标是尽量深入（100层+）
2. **带够食物**：浆果派（+175血量）或购买的高级食物，带20个以上
3. **炸弹是关键**：骷髅洞窟的石头比矿洞更多，炸弹效率极高
4. **运气值影响**：高运气值（电视占卜显示「幸运」）的日子进骷髅洞窟，产出更好
5. **使用楼梯**：遇到多层困难区域，放楼梯直接跳过保存体力
