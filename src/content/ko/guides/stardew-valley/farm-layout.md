## 농장 배치가 중요한 이유

스타듀밸리에서는 무엇이든 어디에나 놓을 수 있습니다. 하지만 즉흥적으로 꾸린 농장보다 미리 설계한 농장이 훨씬 더 효율적입니다. 잘 배치된 농장과 흩어진 농장의 차이는 결국 매일 들이는 플레이 시간으로 환산됩니다.

좋은 배치는 다음을 가능하게 합니다.
- 관련 작업을 한데 모아 이동 시간을 없앤다
- 스프링클러 하나당 덮는 작물 면적을 극대화한다
- 동물 건물을 물가와 서로 가까이 배치한다
- 새 콘텐츠를 해금할 때를 대비해 확장 여지를 남긴다

이 가이드는 먼저 핵심 원칙을 다룬 뒤, 농장 유형별 구체적인 배치를 살펴봅니다.

## 기본 토대: 스프링클러 격자 설계

스타듀밸리에서 가장 중요한 배치 결정은 작물밭을 스프링클러 주변에 어떻게 배열하느냐입니다.

### 고급 스프링클러 격자 (초반~중반)

고급 스프링클러는 3×3 범위(스프링클러 주변 8칸)에 물을 줍니다. 가장 효율적인 배열은 다음과 같습니다.

```
S = 스프링클러, C = 작물 칸, P = 길

P C C C P C C C P C C C
C C C C C C C C C C C C
C C C C C C C C C C C C
P C C C P C C C P C C C
C C C C C C C C C C C C
C C C C C C C C C C C C
P C C C P C C C P C C C
```

각 "S"가 둘러싼 8칸을 덮습니다. 스프링클러 사이의 길은 작물을 밟지 않고 걸어 다닐 수 있게 해줍니다. 이 배열은 작물 칸을 단 한 칸도 낭비하지 않으면서 최소한의 스프링클러만 필요로 합니다.

**중요**: 스프링클러는 작물 칸이 아니라 길 칸 위에 놓으세요. 길로 다니고, 한 칸 건너씩 작물을 심으면 됩니다.

### 이리듐 스프링클러 격자 (후반)

이리듐 스프링클러는 5×5 범위(스프링클러 주변 24칸)에 물을 줍니다. 최적 배열은 다음과 같습니다.

```
가로·세로 6칸마다 스프링클러 한 개
```

이리듐 스프링클러 하나가 5×5 작물 구역을 덮으며, 12칸마다 스프링클러 2개꼴로 깔립니다. 효율이 극도로 높아, 일반 농장의 작물 구역 전체를 이리듐 스프링클러 8~10개로 덮을 수 있습니다.

**전환 시점**: 이리듐 스프링클러를 5개 이상(해골 동굴 드롭, 떠돌이 상인, 또는 농사 레벨 10 직업 효과로) 확보하는 순간, 그에 맞춰 밭 격자를 다시 설계하세요. 효율 도약이 상당합니다.

## 농장 유형별 분석

### 일반 농장 — 가장 많은 작물 칸

일반 농장(기본형)은 가장 넓고 평탄한 무장애 면적을 갖추고 있습니다. 경작 가능한 칸이 약 3,427칸에 달하죠. 작물 수익에 집중하는 플레이어에게 최고의 농장입니다.

**추천 구역 배치**:

**1구역 (중앙/남쪽): 주력 작물밭**
넓은 중앙 구역이 주된 수입원입니다. 시작하자마자 고급 또는 이리듐 스프링클러 격자로 깔아두세요. 계절 작물, 고대 작물(온실용), 고부가가치 작물을 모두 이곳에 심습니다.

**2구역 (서쪽/좌측 상단): 동물 구역**
외양간과 닭장은 물을 얻을 수 있는 연못 근처에 모아 배치하세요. 닭장 1 → 닭장 2 → 큰 닭장 → 디럭스 닭장을 한 줄로 묶고, 이어서 외양간 1 → 큰 외양간 → 디럭스 외양간을 배치합니다. 건초 보관용 사일로도 근처에 두세요. 이 구역은 매일 반복 작업(쓰다듬기, 우유 짜기)을 발생시키므로 한곳에 모아 자급자족하게 만드는 것이 좋습니다.

**3구역 (동쪽): 가공 구역**
통, 저장용 단지, 치즈 압축기 등 장인 가공 기계를 둡니다. 수확 후 동선상에 배치하세요. 매일 작물을 넣고 완성품을 꺼내야 하니까요. 기계를 줄지어 놓은 전용 6×8 구역이 잘 어울립니다.

**4구역 (북쪽 끝): 과일 나무**
과일 나무는 각각 3×3의 무장애 공간이 필요합니다(인접한 나무도, 인접한 물체도 없어야 합니다). 나무 사이에 정확히 2칸씩 비워 줄지어 일찍 심으세요. 다 자라는 데 28일이 걸리지만 제철에는 매일 열매를 맺으니, 1년 차에 5종을 모두 심어 두세요.

**5구역 (농가 근처): 온실**
온실(식료품 저장실 보상)은 매일 들르게 되는 곳에 두어야 합니다. 농가 근처에 배치하면 고대 작물을 돌보는 온실 동선이 아침 루틴의 일부가 됩니다.

### 숲 농장 — 단단한 나무와 채집 중심

숲 농장은 경작 가능한 칸 일부를 포기하는 대신, 다시 자라는 큰 그루터기(단단한 나무 공급원)와 숲 지역에서 생성되는 고유 채집물을 얻습니다.

**얻는 것**:
- 단단한 나무의 정기적 보충(큰 그루터기 재생성) — 비밀의 숲에 접근하지 않는 빌드에 유용합니다
- 숲 지역의 계절 채집물
- 약간 더 흥미로운 지형

**배치 추천**:
숲 농장은 평지가 더 적으므로 작물 밀도를 우선하세요. 확보 가능한 평지에 빽빽한 이리듐 스프링클러 격자를 깝니다. 남서쪽 숲 지역은 과일 나무 과수원으로 활용하세요(고르지 않은 지면에도 나무는 심을 수 있습니다).

**적합한 플레이어**: 채집과 제작 중심 플레이를 즐기는 분. 순수 작물 수익 위주라면 덜 이상적입니다.

---

### 해변 농장 — 독특한 메커니즘

해변 농장은 특정 날씨에 해변과 맞닿은 작물이 물 없이도 자라게 해주지만, 대부분의 스프링클러는 모래 토양에서 작동하지 않습니다(주미노 오두막과 손수 물 주기만 어디서나 통합니다).

**난점**: 스프링클러 최적화입니다. 일반 농장식 격자 접근이 해변 토양에서는 통하지 않습니다. 다음 중 하나를 택해야 합니다.
- 초반에는 물뿌리개를 쓴다
- 스프링클러 호환 칸을 신중하게 골라 배치한다
- 물 주기 없이 수확하는 주미노 오두막을 서둘러 마련한다

**해변 농장이 보상하는 것**: 이 농장만의 메커니즘을 중심으로 계획을 짜는 플레이어. 자연스러운 미관, 계절마다 생기는 "공짜 물 주기" 구역, 그리고 색다른 최적화 퍼즐을 즐길 수 있습니다.

**적합한 플레이어**: 새로운 도전을 원하는, 두세 번째 플레이에 임하는 숙련된 스타듀밸리 유저.

---

### 사방 농장 — 협동 플레이 중심

명확한 사분면 구획을 갖춘 4인 협동 플레이용 농장입니다. 각 사분면은 고유한 지형 특성(하나는 작물에 더 유리하고, 하나는 숲에 가깝고 등)을 갖습니다.

**솔로 플레이**: 문제없이 작동합니다. 그저 넓고 다채로운 농장을 갖게 될 뿐입니다. 서로 다른 지형 유형이 주는 시각적 다양성 때문에 솔로 플레이어들이 일부러 고르기도 합니다.

## 주미노 오두막 커버리지 계획

주미노 오두막(마을 회관 완성 후 해금)은 종반의 수확 업그레이드입니다. 오두막을 중심으로 한 17×17 범위 안에서 작물을 자동 수확하는 주미노를 내보냅니다.

**배치 전략**:
1. 필요한 오두막 수를 계산하세요. 일반 농장의 주력 작물 구역은 대략 50×30칸입니다. 17×17 커버리지 범위는 약 290칸을 덮습니다. 범위가 겹치는 오두막 3~4개면 넓은 밭을 감당할 수 있습니다.
2. 커버리지 범위가 심은 구역 전체를 겹쳐 덮도록 오두막을 배치해, 범위 밖에 남는 작물 칸이 없게 하세요.
3. 오두막 자체 공간(5×3)과 주미노가 작물까지 오갈 길을 남겨 두세요.

**중요한 제약**:
- 비 오는 날에는 주미노가 일하지 않는다
- 겨울에는 주미노가 일하지 않는다
- 다 자라지 않은 작물은 수확하지 않는다
- 수확물은 오두막의 보관 상자에 넣어 두니 정기적으로 회수해야 한다

작물밭 위에 주미노 오두막을 배치하고 나면, 아침 루틴이 20분 넘는 수확에서 오두막 상자를 비우는 2~3분으로 줄어듭니다.

## 목표별 농장 배치 팁

### 목표가 수익 극대화라면

1. 일반 농장의 평지 전체를 작물밭으로 채운다(이리듐 스프링클러 격자)
2. 3년 차까지 주미노 오두막 3~4개로 밭 전체를 덮는다
3. 온실을 얻은 첫날에 고대 작물로 가득 채운다
4. 가공 구역에 통을 50~100개 배치한다 — 와인은 작물 가치를 ×3으로 불린다
5. 매일 송로버섯 수입을 위해 돼지를 들이고, 돼지 4마리 이상을 위해 디럭스 외양간 ×2를 짓는다

### 목표가 미관이라면

1. 더 흥미로운 지형을 위해 숲 농장이나 해변 농장을 고른다
2. 엄격한 격자 대신 곡선과 불규칙한 형태로 작물밭을 설계한다
3. 울타리로 구역을 나눠 별개의 "방"(텃밭, 과수원, 초원)을 만든다
4. 작물 사이에 꽃을 심어 색감을 더한다
5. 비어 있는 구석에 장식물(가구, 조형물, 표지판)을 짓는다

### 목표가 완성(마을 회관)이라면

1. 각 계절 파종 때 꾸러미에 필요한 작물 종류를 우선한다
2. 동물 건물을 일찍 짓는다 — 오리 깃털과 토끼발은 모으는 데 몇 달이 걸린다
3. 밭 근처에 전용 "꾸러미" 상자를 두고 필요한 아이템이 나올 때마다 모은다
4. 1년 차에 과일 나무 배치를 계획한다 — 장인 꾸러미에는 5종 모두가 필요하다

## 계절별 다시 심기 리듬

잘 설계된 농장은 명확한 계절 리듬으로 돌아갑니다.

**봄 1일**: 봄 작물 꾸러미용 작물 + 딸기(달걀 축제 13일 이후) 심기
**봄 1~12일**: 새 인프라 구축(외양간, 통, 스프링클러 배치)
**여름 1일**: 블루베리(주력 밭) + 홉(통 중심) 심기
**가을 1일**: 크랜베리 + 호박(꾸러미) + 옥수수(여름·가을에 걸쳐 자람) 심기
**겨울**: 밭 작물은 없음. 이 시간을 채광, 낚시, 건설, 온실 관리에 활용한다

배치가 명확한 농장에서는 이 계절 리듬이 수월해집니다. 밭이 미리 정돈돼 있으니 어떤 작물을 어디에 심을지 정확히 알 수 있으니까요.
