## Warum das Farm-Layout entscheidend ist

In Stardew Valley kannst du alles überall platzieren – aber eine geplante Farm ist um Welten effizienter als eine zusammengewürfelte. Der Unterschied zwischen einer durchdacht angelegten Farm und einem Chaos misst sich in Stunden täglicher Spielzeit.

Ein gutes Layout:
- spart Laufwege, indem es zusammengehörende Tätigkeiten bündelt
- holt die maximale Anbaufläche pro Sprinkler heraus
- platziert Tiergebäude nah am Wasser und nah beieinander
- lässt Platz zum Erweitern, wenn du neue Inhalte freischaltest

Dieser Guide deckt zuerst die Grundprinzipien ab und danach konkrete Layouts für jeden Farm-Typ.

## Das Fundament: das Sprinkler-Raster

Die wichtigste Layout-Entscheidung in Stardew Valley ist, wie du deine Anbaufelder rund um deine Sprinkler anordnest.

### Qualitäts-Sprinkler-Raster (frühes bis mittleres Spiel)

Qualitäts-Sprinkler bewässern eine 3×3-Fläche (8 Kacheln rund um den Sprinkler). Die effizienteste Anordnung:

```
S = Sprinkler, C = Anbaukachel, P = Weg

P C C C P C C C P C C C
C C C C C C C C C C C C
C C C C C C C C C C C C
P C C C P C C C P C C C
C C C C C C C C C C C C
C C C C C C C C C C C C
P C C C P C C C P C C C
```

Jedes „S" deckt die 8 umliegenden Kacheln ab. Die Wege zwischen den Sprinklern erlauben dir, zu laufen, ohne Pflanzen zu zertrampeln. Diese Anordnung verschwendet keine einzige Anbaukachel und kommt mit der minimalen Anzahl an Sprinklern aus.

**Wichtig**: Setze deine Sprinkler auf die Wegkacheln, nicht auf die Anbaukacheln. Laufe auf den Wegen, bepflanze jede zweite Kachel.

### Iridium-Sprinkler-Raster (spätes Spiel)

Iridium-Sprinkler bewässern eine 5×5-Fläche (24 Kacheln rund um den Sprinkler). Die optimale Anordnung:

```
Sprinkler alle 6 Kacheln horizontal und vertikal
```

Ein Iridium-Sprinkler deckt eine 5×5-Anbaufläche ab, mit 2 Sprinklern auf je 12 Kacheln. Das ist extrem effizient – die gesamte Anbaufläche einer Standard-Farm lässt sich mit 8 bis 10 Iridium-Sprinklern abdecken.

**Wann umsteigen**: Sobald du 5 oder mehr Iridium-Sprinkler hast (aus Funden in der Schädelhöhle, vom fahrenden Händler oder über die Landwirtschaft-Profession auf Stufe 10), gestaltest du dein Feldraster um sie herum neu. Der Effizienzsprung ist enorm.

## Die Farm-Typen im Überblick

### Standard-Farm – die meisten Anbaukacheln

Die Standard-Farm (Voreinstellung) hat die größte zusammenhängende ebene Fläche – rund 3.427 bepflanzbare Kacheln. Das ist die beste Farm für Spieler, die auf Ernteeinnahmen setzen.

**Empfohlene Layout-Zonen**:

**Zone 1 (Mitte/Süden): Hauptfeld**
Die große zentrale Fläche ist deine wichtigste Einnahmequelle. Lege sie sofort als Qualitäts- oder Iridium-Sprinkler-Raster an. Hier kommen deine saisonalen Pflanzen, Antikfrüchte (Gewächshaus) und besonders wertvollen Kulturen hin.

**Zone 2 (Westen/oben links): Tierbereich**
Stelle deine Ställe und Hühnerställe gebündelt nah an einen Teich, damit Wasser in der Nähe ist. Gruppiere sie: Hühnerstall 1 → Hühnerstall 2 → Großer Hühnerstall → Luxus-Hühnerstall in einer Reihe, dann Stall 1 → Großer Stall → Luxus-Stall. Silo(s) in der Nähe für die Heulagerung. Halte diese Zone in sich geschlossen – sie erzeugt tägliche Aufgaben (Streicheln, Melken), und die möchtest du gebündelt haben.

**Zone 3 (Osten): Verarbeitungsbereich**
Fässer, Einmachgläser, Käsepressen und andere Kunsthandwerksmaschinen. Platziere sie dort, wo du nach der Ernte sowieso langläufst – täglich legst du Pflanzen hinein und holst fertige Waren heraus. Ein eigener Bereich von 6×8 mit Maschinen in Reihen funktioniert gut.

**Zone 4 (Hoher Norden): Obstbäume**
Obstbäume brauchen jeweils eine freie 3×3-Fläche (keine benachbarten Bäume, keine benachbarten Objekte). Pflanze sie früh in Reihen mit genau 2 leeren Kacheln zwischen den Bäumen. Sie brauchen 28 Tage bis zur Reife, tragen dann aber in der Saison täglich Früchte – pflanze alle 5 Baumarten in Jahr 1.

**Zone 5 (nahe am Bauernhaus): Gewächshaus**
Das Gewächshaus (Belohnung der Vorratskammer) sollte dort stehen, wo du täglich vorbeikommst. Platziere es nahe am Bauernhaus – der Gang zum Gewächshaus für die Antikfrüchte wird Teil deiner Morgenroutine.

---

### Wald-Farm – Fokus auf Hartholz und Sammeln

Die Wald-Farm opfert einige bepflanzbare Kacheln für nachwachsende Große Baumstümpfe (Hartholzquellen) und einzigartige Sammelobjekte, die in den bewaldeten Bereichen erscheinen.

**Was du bekommst**:
- regelmäßig nachwachsendes Hartholz (Große Baumstümpfe wachsen nach) – nützlich für Durchgänge ohne Zugang zum Geheimen Wald
- saisonale Sammelobjekte im Waldbereich
- etwas abwechslungsreicheres Gelände

**Layout-Empfehlung**:
Da die Wald-Farm weniger ebene Fläche hat, hat die Anbaudichte Priorität: ein enges Iridium-Sprinkler-Raster auf dem verfügbaren ebenen Land. Nutze den bewaldeten Bereich im Südwesten als Obstgarten (der unebene Boden trägt trotzdem Bäume).

**Am besten geeignet für**: Spieler, die gern sammeln und auf kunsthandwerklich geprägte Durchgänge stehen. Weniger optimal, wenn es rein um Ernteeinnahmen geht.

---

### Strand-Farm – einzigartige Mechaniken

Die Strand-Farm lässt strandnahe Pflanzen bei bestimmtem Wetter ohne Gießen wachsen, doch die meisten Sprinkler funktionieren auf dem Sandboden nicht (nur Junimo-Hütten und Handgießen wirken überall).

**Die Herausforderung**: die Sprinkler-Optimierung – der Rasteransatz der Standard-Farm funktioniert auf Strandboden nicht. Du musst entweder:
- im frühen Spiel die Gießkanne benutzen
- die wenigen sprinklertauglichen Kacheln sorgfältig nutzen
- so schnell wie möglich zu Junimo-Hütten kommen, die ohne Gießen ernten

**Wofür die Strand-Farm belohnt**: Spieler, die um ihre einzigartigen Mechaniken herum planen – die natürliche Optik, die saisonalen „Gratis-Gieß"-Zonen und ein ganz neues Optimierungsrätsel.

**Am besten geeignet für**: erfahrene Stardew-Spieler im zweiten oder dritten Durchgang, die eine neue Herausforderung suchen.

---

### Vier-Ecken-Farm – auf Koop ausgelegt

Konzipiert für den 4-Spieler-Koop mit klaren Quadranten-Aufteilungen. Jeder Quadrant hat eigene Geländemerkmale (einer ist anbaufreundlicher, einer waldartiger usw.).

**Solo-Spiel**: funktioniert problemlos – du hast dann eben eine große, abwechslungsreiche Farm. Viele Solo-Spieler wählen sie wegen der optischen Vielfalt der verschiedenen Geländetypen.

## Die Junimo-Hütten-Abdeckung planen

Junimo-Hütten (freigeschaltet nach Abschluss des Gemeinschaftszentrums) sind das Ernte-Upgrade fürs Endgame. Sie schicken Junimos los, die Pflanzen in einer 17×17-Fläche rund um die Hütte automatisch ernten.

**Platzierungsstrategie**:
1. Berechne, wie viele Hütten du brauchst: Das Hauptfeld der Standard-Farm ist grob 50×30 Kacheln groß. Ein 17×17-Abdeckungskreis deckt rund 290 Kacheln ab. 3 bis 4 Hütten mit überlappender Abdeckung bewältigen ein großes Feld.
2. Platziere die Hütten so, dass sich ihre Abdeckungskreise mit deiner gesamten Anbaufläche überschneiden – keine Anbaukachel sollte außerhalb der Reichweite liegen.
3. Lass Platz für die Hütte selbst (5×3) und für den Weg der Junimos zu den Pflanzen.

**Wichtige Einschränkungen**:
- Junimos arbeiten an Regentagen nicht
- Junimos arbeiten im Winter nicht
- sie ernten keine Pflanzen, die noch nicht voll ausgewachsen sind
- sie legen das Erntegut in die Vorratstruhe der Hütte – leere sie regelmäßig

Sobald die Junimo-Hütten über deinem Anbaufeld stehen, schrumpft deine Morgenroutine von über 20 Minuten Ernten auf 2 bis 3 Minuten Einsammeln aus den Hüttentruhen.

## Layout-Tipps nach Ziel

### Wenn dein Ziel maximale Einnahmen sind

1. Fülle die gesamte ebene Fläche der Standard-Farm mit Anbaufeldern (Iridium-Sprinkler-Raster)
2. Platziere bis Jahr 3 3 bis 4 Junimo-Hütten, die das ganze Feld abdecken
3. Fülle das Gewächshaus an dem Tag, an dem du es bekommst, mit Antikfrüchten
4. Stelle 50 bis 100 Fässer in deine Verarbeitungszone – Wein vervielfacht den Pflanzenwert ×3
5. Halte Schweine für tägliche Trüffel-Einnahmen; baue 2× Luxus-Stall für 4 oder mehr Schweine

### Wenn dein Ziel ästhetische Schönheit ist

1. Nimm die Wald- oder Strand-Farm für interessanteres Gelände
2. Gestalte Anbaufelder in geschwungenen, unregelmäßigen Formen statt in strengen Rastern
3. Setze Zäune ein, um deutliche „Räume" zu schaffen (Küchengarten, Obstgarten, Wiese)
4. Pflanze Blumen zwischen die Kulturen für Farbe
5. Stelle Dekoobjekte (Möbel, Skulpturen, Schilder) in ungenutzte Ecken

### Wenn dein Ziel der Abschluss ist (Gemeinschaftszentrum)

1. Priorisiere die Pflanzenarten, die für die Bündel der jeweiligen Saison gebraucht werden
2. Stelle Tiergebäude früh auf – Entenfedern und Hasenpfoten dauern Monate, bis man sie erhält
3. Halte eine eigene „Bündel"-Truhe nahe am Feld bereit, um benötigte Gegenstände zu sammeln, sobald sie auftauchen
4. Plane die Platzierung der Obstbäume in Jahr 1 – für das Kunsthandwerk-Bündel werden alle 5 Arten gebraucht

## Der saisonale Neupflanz-Rhythmus

Eine gut durchdachte Farm läuft nach einem klaren saisonalen Rhythmus:

**Frühling Tag 1**: Pflanze die Kulturen des Frühlingspflanzen-Bündels + Erdbeeren (nach dem Eierfest an Tag 13)
**Frühling Tag 1–12**: Errichte neue Infrastruktur (Ställe, Fässer, Sprinkler-Platzierungen)
**Sommer Tag 1**: Pflanze Blaubeeren (Hauptfeld) + Hopfen (für die Fässer)
**Herbst Tag 1**: Pflanze Preiselbeeren + Kürbisse (Bündel) + Mais (saisonübergreifend)
**Winter**: keine Feldkulturen. Nutze die Zeit für Bergbau, Angeln, Bauen und die Pflege des Gewächshauses.

Eine Farm mit klarem Layout macht diesen saisonalen Rhythmus mühelos – du weißt genau, wo jede Pflanze hinkommt, weil die Felder bereits vororganisiert sind.
